Marketplace static demo

How to preview locally:

- Use any static server. Example with Python 3:

```bash
python -m http.server 8000
```

Open http://localhost:8000 in your browser.

Notes:
- Built with HTML + Tailwind CDN + Vanilla JS
- Products in `assets/scripts/products.js`
- Cart and favorites are persisted in LocalStorage
- To deploy: push to Vercel or any static hosting
